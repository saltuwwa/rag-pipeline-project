# RAG Pipeline Project

Проект по построению RAG-пайплайна для годовых отчётов казахстанских компаний (КТЖ, Матен Петролеум).

## Структура

| Папка | Задание | Содержимое |
|-------|---------|------------|
| **rag-pipeline** | 1 | Построение RAG-пайплайна: парсинг, эмбеддинги, retrieval, генерация |
| **rag-evaluation** | 2 | Эксперименты с гиперпараметрами + RAGAS |
| **graph-rag** | 3 (бонус) | GraphRAG с Neo4j |
